備註

Hugo建構教學

Windows系統需要先安裝CHOCOLATEY套裝管理器

1

Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))

安裝完成後執行choco如有得到相關版本資訊則代表安裝完成，例如：

1
2
3

choco 
Chocolatey v1.2.0
Please run 'choco -?' or 'choco <command> -?' for help menu.

確認安裝完後就可以開始安裝Hugo了

1

choco install hugo -confirm
Mac安裝指令
$ brew install hugo

上述步驟執行完後就完成基本的安裝了｡:.ﾟヽ(*´∀`)ﾉﾟ.:｡

Step2.建立Blog網站

首先先下Command移到所想要新增網站的位置 再執行網站的新增

1
2

CD anyfolder 
hugo new site anywebsitename  ## 建立anywebsitename網站資料夾

Hugo會幫你把所有預設所需要的資料夾都建立出來

接下來就可以找我們所想要的佈景主題了

Hugo Themes Hugo內提供了相當多的網站模板

接下來就是套用我們所喜歡的模板了

筆者是選擇LoveIt這款模板

先將下Command移到網站資料夾下 再將模板給下載至網站名稱/Themes裡

1
2

CD anywebsitename ## 先移至網站資料夾下 
git clone https://github.com/dillonzq/LoveIt.git themes/LoveIt  ## 將整個模板複製到網站資料夾中的Themes

模板下載完後必須先修改網站資料夾中的config.toml

預設內容應該是空白的

我們就加入幾行語法

1
2
3
4

baseURL = 'https://使用者名稱.github.io/'  ## 這個是到時候將網站發布至GitHub會用到的預設URL
languageCode = 'zh-tw'  ## 網站語系
title = "任意修改"  ## 網站Title
theme = 'LoveIt'  ## 網站使用主題
設定網站主題技巧

必須和網站名稱/Themes內的主題資料夾名稱相同

否則會出現錯誤

step3.加入文章吧!!( ﾟ∀ﾟ)o彡ﾟ

完成上述的步驟的話，恭喜你已經完成將近50%的部落格建構了

接下來讓我們來新增文章吧

一樣在網站資料夾下執行新增文章指令

1

hugo new posts/任意檔名.md
文章檔名須注意重點

檔名命名原則盡量以小寫英文及下底線進行命名

未來在建立圖檔連結時比較方便
